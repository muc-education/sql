## Mysql安装

MySQL Community Server,



### Windows上安装

[官网Mysql](http://dev.mysql.com/downloads/mysql/)

下载对应版本（社区版），进行安装。（在线安装版，离线安装版，解压缩版）

### Linux 安装

> (Centos)

``` bash
sudo rpm -ivh mysql57-community-release-el7-9.noarch.rpm

sudo yum install mysql-server
```

### Mac 安装

```bash
brew install mysql
```
